Flashing Instructions

Flash ESP32 via Browser

  1. Connect your device via USB
  2. Click Enter Download Mode below (or hold BOOT manually)
  3. Click Install Firmware on the card above
  4. Select the serial port in the browser popup
  5. Wait for the flash to complete (~30 seconds)
  6. The device will reboot and start automatically
  7. Use the Serial Console below to verify boot output

Requires Chrome, Edge, or Opera. If your device isn't detected, install CP2102 drivers.

Manual fallback: If the button doesn't work, hold BOOT, press RESET, release BOOT to enter download mode, then click Install. The port may change — reselect it. After flashing, press RESET to boot normally.

Flash nRF52 via UF2

  1. Click Download UF2 on the card above
  2. Connect your device via USB
  3. Double-tap the RESET button quickly (~200ms between taps)
  4. A USB drive named XIAO-SENSE or RAK4631 appears
  5. Drag and drop the .uf2 file onto the drive
  6. The drive disappears -- device is flashing
  7. After ~5 seconds, the device reboots automatically
  8. Use the Serial Console to verify: you should see [NEXUS] boot messages
WIO-SX1262 kit: The kit has a single RESET button. Double-tap it quickly to enter UF2 bootloader (a USB drive appears). After flashing, the RED LED blinks once (boot), BLUE blinks 5 times (init), then GREEN blinks 3 times (ready). Rapid RED blinking = LoRa radio not detected.

Works in any browser / OS. The UF2 bootloader is pre-installed on all Adafruit-based nRF52840 boards. No drivers needed. If the USB drive does not appear, try double-tapping RESET faster.

Flash via Command Line

  1. Install PlatformIO CLI
  2. Clone the repository: git clone https://github.com/idan2025/nexus-protocol.git
  3. Navigate to your target: cd firmware/<target>
  4. Build and flash: pio run -t upload
  5. Monitor serial: pio device monitor -b 115200

Targets: heltec_v3, xiao_esp32s3, rak4631, xiao_nrf52840
