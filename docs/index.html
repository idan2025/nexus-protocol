Flashing Instructions

Flash ESP32 via Browser

  1. Connect your device via USB
  2. Click Install Firmware on the card above
  3. Select the serial port in the browser popup
  4. Wait for the flash to complete (~30 seconds)
  5. The device will reboot and start automatically

Requires Chrome, Edge, or Opera. If your device isn't detected, install CP2102 drivers or hold BOOT while plugging in USB.

Flash nRF52 via UF2

  1. Click Download UF2 on the card above
  2. Connect your device via USB
  3. Double-tap the RESET button quickly
  4. A USB drive named XIAO-SENSE or RAK4631 appears
  5. Drag and drop the .uf2 file onto the drive
  6. The device reboots and starts automatically

Works in any browser / OS. The UF2 bootloader is pre-installed on all Adafruit-based nRF52840 boards. No drivers needed.

Flash via Command Line

  1. Install PlatformIO CLI
  2. Clone the repository: git clone https://github.com/idan2025/nexus-protocol.git
  3. Navigate to your target: cd firmware/<target>
  4. Build and flash: pio run -t upload

Targets: heltec_v3, xiao_esp32s3, rak4631, xiao_nrf52840
